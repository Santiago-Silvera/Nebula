# NEBULA Engine Roadmap

This is an estimation of the features for each release version of the engine. Specific features might be moved around depending on the complexity.

## Release 0.1.0

- [x] Mesh
  - [x] Mesh creation
    - [x] VAO
    - [x] VBO
    - [x] EBO
    - [x] Shader
      - [x] Shader loading
      - [x] Shader compiling
- [x] Lighting
  - [x] Diffuse light
  - [x] Ambient light
  - [x] Specular light
    - [x] Specular maps
- [x] Logging
  - [x] Log to stdout

## Release 0.2.0

  - [ ] Options
  - [ ] Cleanup functions
- [ ] Util
  - [ ] File path resolutions
- [ ] Mesh model loading
- [ ] Log to files
